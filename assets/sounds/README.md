# Sound Files

Add the following sound files to this directory:

- `capture.mp3` - Sound when capturing a piece
- `check.mp3` - Sound when giving check
- `error.mp3` - Sound for wrong move (bad move/blunder)
- `already.mp3` - Sound when move already found

## Recommended Sources

Free chess sounds can be found at:
- Lichess sounds: https://github.com/lichess-org/lila/tree/master/public/sound
- Freesound.org (search for "chess")

## Format

- MP3 format recommended for best browser compatibility
- Keep files small (< 50KB each)
- Duration: 0.3-1 second
